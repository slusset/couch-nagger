## Using This Documentation on GitHub Wiki

These documentation files can be easily copied to your GitHub wiki:

1. Go to your repository wiki: https://github.com/slusset/couch-nagger/wiki
2. Click "Create the first page" or "New Page"
3. Copy the content from each `.md` file in this directory
4. Create a new wiki page for each documentation file
5. Use the same filenames (without .md) for wiki page names

The internal links will work automatically in the wiki.
